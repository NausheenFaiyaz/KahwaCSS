KahwaCSS Docs

KahwaCSS is a lightweight JavaScript utility engine. You write kw-* classes in HTML; KahwaCSS parses and applies inline styles.

Installation

npm i kahwacss

Initialize

import { initKahwaCSS } from "kahwacss";

initKahwaCSS();

CDN

<script src="https://cdn.jsdelivr.net/npm/kahwacss@latest/dist/kahwacss.cdn.min.js"></script>
<script>window.KahwaCSS.init();</script>

How The Library Works

  1. 1. The engine scans the DOM for classes starting with kw-.
  2. 2. Each class is parsed into a style object, for example kw-p-4 -> { padding: "16px" }.
  3. 3. Styles are attached using inline style assignment.
  4. 4. A MutationObserver listens for new nodes and class changes and reapplies styles.
  5. 5. Prefix support is intentionally limited to kw- only.

Supported Utility Groups

Spacing

kw-p-*, kw-pt-*, kw-px-*, kw-m-*, kw-mx-auto, arbitrary values like kw-m-[12px].

Colors and Typography

kw-bg-red, kw-bg-[#111], kw-text-gray, kw-text-4xl, kw-font-semibold, kw-lh-24.

Sizing

kw-w-full, kw-w-1/2, kw-h-screen, kw-min-w-320, kw-max-h-500.

Layout and Flex/Grid

kw-flex, kw-items-center, kw-justify-between, kw-grid-cols-3, kw-gap-4, kw-grow-1.

Borders and Effects

kw-border-1, kw-border-b-2, kw-border-color-blue, kw-rounded-2xl, kw-shadow-md, kw-opacity-80.

Interaction and Other

kw-transition, kw-duration-300, kw-pointer-none, kw-object-cover, kw-overflow-hidden, kw-z-50.
