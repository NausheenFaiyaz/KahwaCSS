Features

Runtime Styling

Apply styles directly from HTML class tokens without generating a stylesheet.

Mutation Aware

New nodes and class updates are observed, parsed, and styled automatically.

Single Prefix

Strict kw- prefix keeps your utility syntax focused and conflict-free.

How It Works

  1. 1. KahwaCSS scans elements containing classes starting with kw-.
  2. 2. Tokens are parsed into JavaScript style objects using utility rules.
  3. 3. Styles are applied inline and re-applied when DOM classes change.
